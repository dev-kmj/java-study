# 📌 상속 & 오버라이딩

## 상속의 이해
### 상속의 작동 방식
**한 클래스가 다른 클래스를 상소갛는 것을 하위 클래스가 상위 클래스로부터 상속받는다**고 말함
자바에서는 **하위 클래스가 상위 클래스를 확장(extend)한다**고 얘기함
하위 클래스가 상위클래스의 멤버를 물려받는다는 것을 의미

예를 들어,

- 상위 클래스 (더 추상적)
  **SuperHero**
  
  |인스턴스 변수|
  |-------------|
  |suit|
  |tights|
  |specialPower|

  |메서드|
  |-------|
  |useSpecialPower()|
  |putOnSuit()|

- 하위 클래스 (더 구체적)
  **FriedEggMan**
  
  |   |
  |-------------|

- 오버라이드하는 메서드
  **PantherMan**

  |메서드|
  |-------|
  |useSpecialPower()|
  |putOnSuit()|
