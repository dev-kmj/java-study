# 클래스와 객체

- 객체가 아는 것 : 인스턴스 변수 (상태)
- 객체가 하는 것 : 메서드 (행동)

|ShoppingCart|     
|------------|
|cartContents|
|<span style="color:blue">addToCart()</span>|
|RemoveFromCart()|
|checkOut()|

- 아는 것 : cartContents
- 하는 것 : addToCart(), RemoveFromCart(), checkOut()
