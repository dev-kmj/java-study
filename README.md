# 📖 자바 스터디

## 📢 진행방식 
* 일정 : 2024.03.05(화) ~ 
* 매주 `화, 금요일 3:00pm`까지 각자 공부한 내용을 업로드합니다.
* 원본 저장소에 디렉토리 생성 후 각자 fork한 저장소를 업데이트한 뒤 내용을 작성합니다. 
* 팀원의 자료를 보면서 궁금한 점이나 새로 알게 된 부분, 질문과 답변은 issue에 자유롭게 기록합니다.
* 스터디 진행 주제는 `헤드 퍼스트 자바`책을 기준으로 필요한 부분을 골라 선정했습니다.
